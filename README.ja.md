# PyQt6 画像データ収集・管理システム (Pixivダウンローダー)

> Language / 语言: [English](README.md) |[简体中文](README.zh-CN.md)

**PyQt6** と **Python** を利用して開発された Windows デスクトップアプリケーションです。特定の Pixiv アーティストの画像データを自動収集し、ローカルの **SQLite** データベースで管理する機能を実現します。

元々は MySQL を使用していましたが、より軽量でポータブルな運用（サーバーレス）を実現するため、SQLite を使用して再構築しました。

MySQLのレポジトリ: <https://github.com/yexca/PixivDownloader-MySQL>

## 主な機能

* **ユーザーフレンドリーなGUI:** PyQt6 で構築されたシンプルでクリーンなインターフェース。
* **データ自動収集:** 指定したアーティストの作品（画像）を自動的にダウンロード。
* **ローカルDB管理:** 収集したメタデータ（ID、URL等）は、ローカルの SQLite データベースに保存・整理されます。
* **軽量・ポータブル:** SQLite を採用することで、別途データベースサーバーを必要とせず、手軽に利用・バックアップが可能です。

## 技術スタック

* **Python 3**
* **PyQt6:** デスクトップGUI用。
* **SQLite3:** 軽量なローカルデータベース用。

## 実行方法

1.  **リポジトリをクローン:**
    ```bash
    git clone https://github.com/yexca/PixivDownloader-SQLite.git
    cd PixivDownloader-SQLite
    ```

2.  **依存関係のインストール:**
    ```bash
    pip install PyQt6
    ```

3.  **アプリケーションの実行:**
    ```bash
    python main.py
    ```
